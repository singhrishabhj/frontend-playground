HTML Anchor Tag with Download Attribute

The < a > tag in HTML, commonly known as the anchor tag, is used to create hyperlinks. One of its useful attributes is download, which instructs the browser to download the resource linked in the href attribute instead of navigating to it.

Code Downloads

Attributes Breakdown

1. href : This attribute specifies the URL of the file you want to link to. It can be a path to a file on the web or a local file relative to the HTML file.

2. download : When present, this attribute tells the browser to download the file instead of opening it. The value of the download attribute, if provided, will be the new file name of the downloaded file. If no value is given, the original file name is used.

< a href="./image.jpg" download="MyImage.jpg">Download Image< /a >
* In this case, clicking the link will download the file as MyImage.jpg.

HTML Lists

- In HTML, lists are used to group a set of related items.

- There are two main types of lists: unordered lists and ordered lists. Below is a breakdown of the provided code, explaining each part and its purpose.

Let's talk about :

1. UNORDERED LIST (< ul >)

An unordered list is a collection of items where the order does not matter. Each item in the list is marked with a bullet point by default.

Unordered list

  • Apple
  • Oranges
  • Grapes
  • Water Melon
  • Coffee

Explanation:

Unordered list:

This is a heading element (<h4>) that titles the section as "Unordered list". It is not part of the list itself but serves as a label.

<ul>: This tag defines the start of an unordered list.

<li>Apple</li>: Each <li> tag defines an individual list item. In this case, "Apple" is an item in the unordered list.

The list continues with more <li> tags for "Oranges", "Grapes", "Water Melon", and "Coffee".

</ul>: This tag defines the end of the unordered list.

2. ORDERED LIST (< ol >)

An ordered list is a collection of items where the order does matter. Each item in the list is marked with a number by default.

Ordered list

  1. HTML
  2. CSS
  3. JS
  4. REACT
  5. NEXTJS

Explanation:

<h2>Ordered list</h2>: This is a heading element (<h2>) that titles the section as "Ordered list". It is not part of the list itself but serves as a label.

<ol>: This tag defines the start of an ordered list.

<li>HTML</li>: Each <li> tag defines an individual list item. In this case, "HTML" is an item in the ordered list.

The list continues with more <li> tags for "CSS", "JS", "REACT", and "NEXTJS".

</ol>: This tag defines the end of the ordered list.
